# Lista de Compras Java

Projeto simples de lista de compras feito em Java.

## Funcionalidades

- Adicionar itens
- Ver lista
- Remover itens
- Encerrar programa

## Tecnologias utilizadas

- Java
- IntelliJ IDEA
- Git
- GitHub

## Como executar

1. Abrir no IntelliJ
2. Executar a classe `ListaCompras`
3. Usar o menu do terminal
